Release Description

The first public beta version of the KyBook new edition. This version is still unfinished and contains lots of errors and stubs for not yet implemented features. The release is intended for your familiarization with the new design of KyBook. Major innovations: new engines for epub/fb2 ebook formats. Heavily modified engines for pdf/djvu. A new way for cataloging books and cloud storages interaction.
