Release Description

  • Added settings for a bookshelf.
  • Added the ability to share the auto summarization result.
  • Added the feature to save a book page into Photo Stream.
  • Added support for printing a book page.
  • Added the stop-timer for audiobooks.
  • Added a setting for auto opening a book at the application launch.
  • Added vertical scrolling without pagination (roll) in fb2 and epub (v2) books.

  • Improved support of table of contents in DJVU books.
  • Improved performance in PDF books.
  • Improved rendering of tables in fb2 and epub (v2) books

  • Fixed crash in PDF books when searching.
  • Fixed various minor bugs.
